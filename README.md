# Mindi Royale

A premium Mindi (Mendikot) multiplayer card game web application built with Vue 3.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

## Build

```bash
npm run build
```
